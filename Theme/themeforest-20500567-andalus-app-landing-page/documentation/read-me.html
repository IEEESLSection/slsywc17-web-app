Introduction

Andalus is a powerful Responsive App Landing Page template based on Bootstrap. It is 100% responsive and looks stunning on all types of screens and devices. This template has a universal design, it thought every detail and animation effect. Its easy to customize to fit your needs just replace images and texts and Its offers many customization options. The template comes with multiple homepages variations and pages. Andalus is a great choice to present your app ideas and promote your app

CSS Files and Structure

I'm using latest CSS coding standard. Custom Style written into style.css which is override Bootstrap style somewhere else.

CSS Files included:
  • Bootstrap CSS
  • Font Awesome CSS
  • Lity CSS
  • Animate CSS
  • Owl Carousel CSS
  • Custom CSS

style.css file contains all of the specific styling for the template. If you would like to edit a specific section of the site, simply it will be in the same order in HTML file, scroll down until you find the appropriate style that needs to be edited.

If you change the style of Features section, find the following section in the stylesheet: 

.feature {
  padding: 30px;
  border-radius: 2px;
  text-align: center;
  -webkit-transition: all 0.3s ease-in-out;
  -o-transition: all 0.3s ease-in-out;
  transition: all 0.3s ease-in-out; }
  .feature img {
    max-width: 60px; }
  .feature h2 {
    font-size: 1.25rem;
    padding: 15px 0; }
  .feature p {
    max-width: 300px;
    font-size: 0.875rem;
    margin: 0 auto; }
  .feature:hover {
    -webkit-box-shadow: 0 0 50px rgba(0, 0, 0, 0.1);
    box-shadow: 0 0 50px rgba(0, 0, 0, 0.1);
    -webkit-transform: translateY(-5px);
    -ms-transform: translateY(-5px);
    transform: translateY(-5px); }

Images

All Images included are free and ready to use, at <section id="gallery"> you don't have to remove the iPhone mockup, only replace the current <img> with your image,

<div class="gallery__item text-center">
  <img src=“img/gallery1.png” alt="App Screenshot">
</div>

All other images are entirely replaceable, The PSD included contains all editable freebies, that makes it easier for you to attain the desirable style.

Javascript

Andalus imports following Javascript files.

  • jQuery 2.1.3 CDN
  • Bootstrap 3
  • Lity
  • jQuery One Page Nav
  • jQuery Easing
  • Owl Carousel
  • WOW
  • Custom Script

All code are beautifully written. You can customize easily. Just look at the js code.

Example: If you want to Configure the team slider, just find the following block and play with the slider options: 

$("#team__slider").owlCarousel({
  autoplay: true,
	autoplayTimeout: 5000,
	loop: true,
	margin: 10,
	nav: false,
	pagination: false,
	dots: true,
	responsiveClass: true,
	responsive: {
			0: {
				items: 1,
			},
			600: {
				items: 3,
			},
			1000: {
				items: 4,
			}
	}
});
